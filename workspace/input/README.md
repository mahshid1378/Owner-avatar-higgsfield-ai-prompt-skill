# input/

Drop anything you want the skill to **read** here: scripts, story bibles,
creative briefs, character sheets, reference notes, brand guidelines, shot lists.

When the skill asks you for a document, place it in this folder. The skill checks
here first and will move any stray uploads from the project root into this folder
before working with them.

Files here are git-ignored (kept local). This README is the only tracked file.
